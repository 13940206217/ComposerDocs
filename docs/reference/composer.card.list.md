# Composer卡片列表

列出存储在本地钱包中的所有业务网络卡片，或者如果指定了卡片，则显示该业务网络卡片的详细信息。
```bash
composer card list --name admin@tutorial-network
```

## 句法
```bash
Options:
  --help         Show help  [boolean]
  -v, --version  Show version number  [boolean]
  --name, -n     The name used to identify the card  [string]
```
